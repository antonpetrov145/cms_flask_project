# CMS Flask Project

CMS Flask Project is a python app that lets you create, edit and delete posts. 

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the dependencies for the CMS.

```bash
pip install -r requirements.txt
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
